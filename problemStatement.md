# TravelMind – AI Travel Planner

TravelMind is an AI-based travel planning app that helps users create a complete trip plan from a simple text input.

Instead of using multiple platforms, users can describe their trip and get a structured itinerary with recommended hotels and activities.

---

## Features

- Convert user input into travel plans
- Recommend hotels and activities
- Generate day-by-day itinerary
- Save items to wishlist

## Tech Stack

- Frontend: React
- Backend: Spring Boot
- Database: MongoDB 


## Goal

To simplify trip planning by combining recommendations, planning, and organization into one platform.
